# The Paper Room

A mobile browser puzzle game inspired by layered paper-and-vine letter puzzles.

Players drag white fragments across the paper board until the hidden three-letter code becomes readable, then enter the code on the side keypad. The game includes 10 completeable puzzles and a reveal button for checking the solved state.

## Run locally

```sh
npm install
npm start
```

Open `http://127.0.0.1:4173`.

## Test

```sh
npm run test:e2e
```
